      Шаблон сайта на Java script.
      
      Если посмотреть HTML код этой страницы, то вы увидите, что он короче и проще кода предыдущих страниц. Здесь нет тэгов <TABLE>, <TR>, <TD>. Вместо этого использованы Javascript, загружаемые из файлов javasite1.js , javasite2.js и т.д.
      
      <script language=javascript src=javasite1.js></script>
      
      Файл javasite1.js имеет следующий вид:
      
      // <!--
      document.write("<table border=0 cellpadding=0 cellspacing=0 width=800>");
      document.write("<tr valign=middle>");
      document.write("<td background=1.jpg width=57 height=97></td>");
      document.write("<td background=2.jpg width=100 height=97></td>");
      document.write("<td background=3.jpg width=60 height=97></td>");
      document.write("<td background=4.jpg width=504 height=97></td>");
      document.write("<td background=5.jpg width=79 height=97></td>");
      document.write("</tr>");
      document.write("</table>");
      // -->
      
      Т.е. все тэги для формирования ячеек таблицы вынесены в Javascript. Это сильно упрощает заполнение шаблона сайта контентом для начинающих пользователей. При этом архив шаблона сайта может вообще не содержать графики и файлов с Java скриптами, а загружать их с другого сайта:
      
      <script language=javascript src=http://www.oflameron.ru/javasite1.js></script>
      
      При этом полностью сохраняется табличная структура шаблона сайта. Можно заменить значения ширины рамки сайта на переменную Java script. Тогда сайт можно легко настраивать под нужный размер.
      В Java script файлы можно вынести и код счетчика и таблицу стилей <style type="text/css">
      Файлы javasite1.js, javasite2.js, javasite3.js, javasite4.js следующие друг за другом можно объединить в один, что еще больше упростит размещение контента в шаблоне.
      Так же можно сгруппировать файлы javasite5.js, javasite6.js, javasite7.js и javasite8.js, javasite9.js, javasite10.js
      
      [Полный листинг сайта]
      
      Листинги файлов Javascript:
      
      [javasite1.js]  [javasite2.js]  [javasite3.js]  [javasite4.js]  [javasite5.js]  [javasite6.js]
      
      [javasite7.js]  [javasite8.js]  [javasite9.js]  [javasite10.js]  [javasite11.js]

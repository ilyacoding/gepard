      Посмотрев информацию о посещаемости нескольких сайтов (если она не закрыта), можно обнаружить значительное количество просмотров сайтов с мобильных устройств - смартфоном, планшетов. Такие устройства имеют некоторые техничесие ограничения по сравнению с "полноразмерными" компьютерами. У них меньше объем оперативной памяти и значительная ее часть занята операционной системой. Используются броузеры с ограниченным функционалом, адаптированные к небольшому объему памяти. Модули для обработки Flash и Java script у таких броузеров могут работать неустойчиво при нескольких открытых сайтах одновременно. Производительность процесоров и графических систем таких устройств ограничена. Поэтому целесообразно использовать сайты с минимальным графическим оформлением, без графического меню, заставок и кнопок и с наиболее простым HTML кодом.

Контент сайта. Размещение контента сайта в шаблоне


      Шаблоны сайтов >>>>  Бесплатный контент >>>>  Java script >>>>
      
      Если вы разрабатываете свой сайт на основе стандартных шаблонов, то вам придется самостоятельно разработать качественный контент сайта и грамотно его разместить. HTML код размещенных здесь шаблонов сайтов хорошо структурирован и документирован и разместить контент овсем не сложно.
      Если вам нужен сайт с готовым контентом - скачайте готовые сайты с контентом.
      
      Структура шаблона
      
      Рассмотрим размещенные на этом сайте шаблоны. Все они относятся к типу табличных шаблонов с фиксированной шириной. Табличные шаблоны имеют очень логичную и понятную структуру - таблицу <table> </table> с строками <tr> </tr> и ячейками <td> </td>.
      Фиксированная ширина задана абсолютными значениями ширины таблицы:
      
      <table border=0 cellpadding=0 cellspacing=0 width=800>
      
      Ширина строки <tr> </tr> в данном шаблоне будет определяться суммарной шириной составляющих ее ячеек <td> </td>:
      
      <td background=18.jpg width=57 height=10></td>
      <td background=19.jpg width=100 height=10 valign=top></td>
      <td background=20.jpg width=60 height=10></td>
      <td background=21.jpg width=504 height=10></td>
      <td background=22.jpg width=79 height=10><td>
      
      Если просуммировать ширину ячеек, получится 57+100+60+504+79=800 - т.е. ширина таблицы <table> </table>
      
      Этот блок HTML кода стандартен для всего шаблона. Отличия только в графическом оформлении фона background=
      
      background=18.jpg -
      
      background=19.jpg -
      
      background=20.jpg -
      
      background=21.jpg -
      
      background=22.jpg -
      
      Т.е. это элементы графического оформления шаблона этого сайта:
      
       - средняя часть шаблона
      
      Вся таблица шаблона сайта состоит из 3-х основных частей - верхней "шапки" (графическая часть с закругленными краями), средней части (самая большая, в ней располагается контент сайта) и нижней (нижняя графическая часть с закругленными краями).
      
      
       - верхняя часть шаблона
      
       - нижняя часть шаблона
      
      В шаблонах сайтов с сложной графикой верхних и нижних частей может быть несколько. Количество ячеек <td> </td> в строках <tr> </tr> и ячейками <td> </td> определяется дизайном и необходимым количеством столбцов для размещения контента. В этом варианте шаблона есть два столбца для контента - шириной 100 и шириной 504 pix.
      В ячейках шириной 100 pix размещены ссылки на сайты по теме. В ячейках шириной 504 pix размещен основной контент сайта.

      Java script для сайта
      
      Размеры таблицы и ячеек легко могут быть пересчитаны под разрешение экрана. Например, 1024 pix. Вертикальный размер таблицы (высота) не фиксирована. Указанное значение height=10 в данном случае весьма формально. По мере добавления контента длина таблицы может быть любой. Т.е. по вертикали шаблон сайта раздвижной.
      Размер экрана можно вычислить и использовать для вычисления ширины таблицы и ячеек.
      
      <script language="JavaScript">
      <!--
      var ts=screen.width;
      var tk=screen.height;
      document.write("Screen resolution="+ts+"x"+tk+" pixels");
      //-->
      </script>
      
      ts - ширина экрана
      tk - высота экрана
      
      Java script для динамического создания таблицы нужных размеров (на 100 pix уже ширины экрана):
      
      <script language="JavaScript">
      <!--
      var ts=screen.width - 100;
      document.write("<table border=0 bgcolor=#C7650E cellpadding=0 cellspacing=0 width="+ts+">");
      document.write("<tr bgcolor=#C7650E><br><font class=gtext>123456789</font>");
      document.write("</tr>");
      document.write("</table>");
      //-->
      </script>
      
      
      Если объем контента очень большой, разместите его в нескольких таблицах, размещенных друг за другом. Такой сайт будет отображаться броузером на экране быстрее. Особенно, если есть внешние Java script (например, счетчики), которые могут грузиться долго.

HTTP/1.1
Поддержка:
- keep-alive

Авторизация:
- Basic auth
- Digest auth (RFC 2069)

Методы:
-> GET (обычный, условный, частичный)
-> HEAD
-> OPTIONS

Логи:
-> Глобальный лог ошибок сервера
-> Access и error лог для каждого виртуального хоста

Конфигурационные файлы:
-> Глобальный конфигурационный файл для сервера (IP, port, keep-alive...)
-> Конфигурационный файл со списком виртуальных хостов (настройка логов, авторизации, директории сайта..)
